About me

Hello! I'm Kieron, a Year 3 student studying Computer Science in NUS. I picked up programming while serving National Service by learning Python and trying to solve Project Euler problems. Since then, I have dabbled in Web Development, Machine Learning and Object Oriented Programming.

This website is a personal project that had been put off for a long time. It allows me to record and reflect on my experiences while learning programming. I promised to complete this site during my Year 1 winter break so here it is.
